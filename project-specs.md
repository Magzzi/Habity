# Habity

- tracks your habits
- mark their completion status
- view your progress over time

## Features
1. **Add/Edit/Delete Habits**
    - input habit names
    - set goals
    - table or list
2. **Mark Completion**
    - mark habits as completed
3. **Progress Visualization**
    - view progress over time
    - chart.js
4. **Get Notified**
    - Notification API or alert system
